# Фронтенд

Пользовательский интерфейс — **статические файлы** в каталоге `static/`: разметка, стили, скрипты на **нативных ES-модулях** (import/export без сборщика).

## Файлы

| Путь | Назначение |
|------|------------|
| `static/index.html` | Разметка: форма создания/редактирования, таблица списка, поиск, toasts, подключение Lucide и `app.js` как модуль |
| `static/css/styles.css` | Внешний вид (тема, сетка, таблица, скелетоны, кнопки) |
| `static/js/app.js` | Логика UI: загрузка списка, отправка формы, редактирование, удаление, поиск, уведомления |
| `static/js/api.js` | Обёртки `fetch` к `/api/contacts` |
| `static/js/utils.js` | Форматирование телефона (маска РФ), даты, вспомогательные DOM-утилиты |

## Базовый URL API

В `api.js` задано `const API_BASE = ""`. Запросы идут на **относительные** пути `/api/...`:

- при локальном запуске Uvicorn с статикой — тот же origin, всё работает;
- в Docker Nginx проксирует `/api/` на backend (см. [docker.md](docker.md)).

Если UI когда-либо разместят на другом домене, потребуется выставить `API_BASE` на URL backend (и настроить CORS).

## Поведение

- **Список:** после загрузки контакты кэшируются в памяти; **поиск** фильтрует кэш без повторных запросов (`normalizeQuery` + сопоставление подстроки по полям).
- **Форма:** скрытое поле `contact-id` переключает режим создания и редактирования; кнопка сброса очищает форму.
- **Ошибки API:** разбор JSON в `parseError` поддерживает формат `{ error: { message } }` и классический FastAPI `detail`.
- **Телефон в таблице:** отображается через `formatPhoneRu` (маска `+7 (…) …` для российских номеров).
- **Дата:** `formatDate` предпочитает разбор префикса `YYYY-MM-DD`, иначе парсит строку как дату.

## Иконки и шрифты

- **Lucide** подключается с CDN в `index.html`, иконки помечаются атрибутом `data-lucide`; после динамического DOM вызывается `lucide.createIcons()`.
- Шрифт **Inter** загружается с Google Fonts.

## Отдача файлов

1. **Локально:** FastAPI монтирует `static/` на `/` при `SERVE_STATIC=1`.
2. **Docker:** статика копируется в образ Nginx (`frontend/Dockerfile`); backend статику не отдаёт (`SERVE_STATIC=0`).
